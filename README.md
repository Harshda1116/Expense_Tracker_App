# Expense Tracker Web Application

A simple web application to track and manage daily expenses. This project helps users record their spending and view expense history through a clean and user-friendly interface.

## Features
- Add daily expenses
- View expense records
- Store expense data in a database
- Simple and responsive UI
- Lightweight Flask backend

## Tech Stack
- Python
- Flask
- HTML / CSS
- SQLite Database

## Project Structure

```
Expense_Tracker_App
│
├── app.py
├── expense.db
├── templates
│   └── index.html
└── README.md
```

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/Harshda1116/Expense_Tracker_App.git
```

2. Navigate to the project folder

```
cd Expense_Tracker_App
```

3. Install dependencies

```
pip install flask
```

4. Run the application

```
python app.py
```

5. Open in browser

```
http://127.0.0.1:5000
```
